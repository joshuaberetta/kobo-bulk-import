GETTING STARTED (3 STEPS)
================================================================================

Step 1: Generate Mapping (one-time)
  python generate_mapping.py your_form.xlsx mapping.json

Step 2: Convert Excel to XML
  python excel_to_kobo_xml.py your_data.xlsx mapping.json output/

Step 3: Submit to Kobo (optional)
  python submit_to_kobo.py \
      --excel your_data.xlsx \
      --mapping mapping.json \
      --form-id YOUR_FORM_ID \
      --token YOUR_API_TOKEN

================================================================================
WHAT DOES THIS DO?
================================================================================

This tool solves a common KoboToolbox workflow problem:

  1. You export data from Kobo to Excel
  2. You process/modify the data in Excel or another tool  
  3. You need to submit the modified data back to Kobo

The challenge: KoboToolbox API requires XML format, but you have Excel.

This toolkit:
  ‚úì Reads your Excel file (with repeat groups on separate sheets)
  ‚úì Rebuilds the proper hierarchical XML structure
  ‚úì Handles repeat groups automatically
  ‚úì Submits to KoboToolbox API (optional)

================================================================================
YOUR DATA STRUCTURE
================================================================================

Your Excel file should look like this:

  Sheet: data                    ‚Üê Main form data
    Columns: today, org_name, ..., _submission__uuid
    
  Sheet: FOCAL_POINTS           ‚Üê Repeat group 1
    Columns: name, email, ..., _submission__uuid
    
  Sheet: FIGURES_COMMUNITY      ‚Üê Repeat group 2  
    Columns: parish, community, ..., _submission__uuid

Key requirement: All sheets MUST have a _submission__uuid column to link 
records together.

NEED HELP?
================================================================================

Common Issues:

  "Column not found in mapping"
    ‚Üí Regenerate mapping.json from your latest XLSForm
    ‚Üí Ensure Excel column names match XLSForm question names

  "No submission found with UUID"  
    ‚Üí Check _submission__uuid exists in all sheets
    ‚Üí Verify UUIDs match across sheets

  API errors (401, 403, 404)
    ‚Üí Check your API token and form ID
    ‚Üí Verify permissions on the form

For more help:
